# TurboAPI

[TurboAPI](https://github.com/justrach/turboAPI) is a drop-in FastAPI replacement with a Zig HTTP core.

Python 3.14 free-threaded + Zig 0.15 native HTTP server. 24-thread pool, zero-alloc response pipeline, response caching.

## Test URLs

* JSON: `http://localhost:8080/json`
* Plaintext: `http://localhost:8080/plaintext`
