# [Zinc](https://zinc.zon.dev) web framework

## Description

Zinc is a web framework written in pure Zig with a focus on high performance, usability, security, and extensibility.

* [Documentation](https://zinc.zon.dev/)

### Some features are:
- **Fast**
- **Custom allocator**
- **Multithreading**
- **Middleware**
- **Routes grouping**
- **Rendering built-in**
- **Extensible**
- **Suite of unit tests**
- **Usability**
