Weber

Getting started

Welcome to the default Weber's page. Weber is a MVC Rails like web framework which was built with Elixir programming language. Build web applications quickly and efficiently as possible. Let's build Web with Elixir.

Weber features:


  • MVC web framework;
  • New project generation;
  • Json generation with exjson;
  • Websocket support;
  • HTML helpers;
  • Site internationalization
  • Sessions support;
  • and many more
